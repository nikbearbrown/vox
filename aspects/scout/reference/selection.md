# Selection: What Makes a Vox-Explainer Film

The scout's job is judgment. Most chapter material does not belong in an
explainer. Keep only concepts that pass the bar below.

**On length:** length is DERIVED from the concept, never chosen. A vox film is
a minute if it honestly fits, typically 3–5 minutes for technical material,
**hard cap 5:00** — a concept whose honest treatment runs past 5:00 is split at
card time into two cards, each with its own Question. Never reject a concept
for being long; never cram one to fit. Teachability, not runtime, is the bar.

## The unit is the concept, not the chapter
Triage by **concept (high-assertion zone)**, never by chapter. A chapter yields
zero, one, or several candidates — all correct. Do not pad a thin chapter, do
not cap a rich one. The scout surfaces the full slate; the human consolidates
at pick-time.

## Step 1 — Find the high-assertion zones (cajal heuristics)
Flag the concepts where a claim is made that prose alone cannot make a reader
*see*. Three detectors, per concept:
- **MC — Mechanism/process:** 3+ interdependent steps or interacting parts.
  The text asserts *how something works*.
- **VG — Verification gap:** a structural/spatial/before-after claim you cannot
  verify from text. The text asserts *how something is arranged*.
- **PQ — Proportional/quantitative:** percentages, ratios, magnitudes,
  distributions. The text asserts *how much*.

## Step 2 — Motion test (does motion carry the teaching?)
Does the learner need to see HOW the transition happens — the mechanism of
change itself — or just the before/after states? Mechanism → film candidate.
States → a static figure teaches it better; drop it (route it to the figures
pipeline, not here). Flag as a candidate if any apply: the transition mechanism
IS the learning target; 3+ sequential causal stages that build; a cyclical
process where return-to-start is the concept; a transformation too fast/slow/
small to observe directly. Motion that adds no instructional meaning is
cognitive load — not a candidate.

## Step 3 — The vox bar
A keeper has ALL of:
1. **A single tension** statable in one sentence with no visual (the Hook).
2. **A key case** — one concrete instance of the mystery you can SHOW in a
   cold open: a specific thing happening, not a category of things.
3. **A Question in gap form** — "X should predict Y. Here is the case where
   it didn't. Why?" If you cannot write this beat, the concept is not ready
   (this is SLATE-RUNNER's mandatory QUESTION beat, caught at scout time).
4. **A single resolution** — one mechanism that dissolves the question
   (the Core idea).
5. **One visual object** the film orbits.
6. **An example seed** — a realistic made-up instance with small countable
   numbers that the builder can walk end to end (THE EXAMPLE act, 16:9).
7. **≤6 elements per scene** (per scene, never per film) and
   **self-containment** — no chapter required, no prior films required.
If a concept needs a derivation, or three prerequisites you'd have to teach
first, or two unrelated visual objects — narrow it to the one surprising slice
or reject it.

## Score rubric (/10) — add points, cap at 10
- **Surprise / "aha" (0–3)** — a genuine "wait, what?" scores 3.
- **Visual obviousness (0–3)** — the idea maps to one drawable object that
  moves; if you can see the animation immediately, 3.
- **Self-containment (0–2)** — stands alone with common prerequisites.
- **Pedagogical payoff (0–2)** — fixes a common misconception or unlocks
  later material.
8–10 = build soon. 6–7 = build after tightening. ≤5 = shelve.

## Manim move vocabulary (the dominant animation verb)
| Move | The object… |
|---|---|
| `compare` | two states side by side, one right / one broken |
| `scan` | a value sweeps across a region |
| `rotate` | spins / orbits / turns on an axis |
| `accumulate` | pieces pile up into a pattern |
| `morph` | one shape continuously becomes another |
| `split` | one thing separates into two |
| `transform` | a quantity sharpens while its partner blurs |
| `slosh` / `spread` | a packet widens, disperses, oscillates |
| `duplicate` | a copy is created matching the original |
| `collapse` | a superposition/coherence decays to one outcome |
| `decay` | amplitude falls off exponentially |
| `trace` | a path or curve is drawn out over time |
If none fit, name the verb plainly — the list is a guide, not a cage.

## Still lanes (seed the SHOTLIST/PROMPTS lane per fill-in still)
- **geo** — abstract mechanism plates built from geometric primitives
  (hand-authored SVG, free, `source: own`). DEFAULT for anything the graphics
  plane can say.
- **c2v** — figurative objects in the flat house style via Illustrator
  Concept-to-Vector (credits; draft cheap, commit once). For things that need
  to *be* a recognizable object but should match the graphics plane.
- **raster** — photographic-feel generation (Midjourney/Higgsfield). LAST
  resort: faces, archival texture, period photographs.

## Exclusion discipline
Exclusions bound the QUESTION, not the length — a 4-minute film still excludes
what its card excludes. Name the specific things a well-meaning builder would
be tempted to add and must not: the derivation, the formalism, the second
example, the historical aside. Generic ("keep it simple") is useless; specific
("no transmission-coefficient derivation, no WKB") is the whole value.
