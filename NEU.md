# NEU — the Northeastern classroom audience

**Audience.** A Northeastern University student inside a specific course, watching a
reel their professor assigned. Not the general mid-career professional (that's
NIKBEARBROWN) — a student with a syllabus, a reading list, and an assessment coming.
NEU mode exists because **NEU professors will use vox for their classes**, and a class
video has to sit inside a course, cite like a course, and look like Northeastern.

**Narration voice.** `ELEVENLABS_VOICE_NEU` if set — **else falls back to
`ELEVENLABS_VOICE_NIKBEARBROWN`** (Bear's voice), which is the default. Bear uses NEU
mode for his own NEU slides, so his voice ships as the default and NEU works out of the
box. **Other professors must create or choose their own `ELEVENLABS_VOICE_NEU`** — Bear's
voice is distinctive enough that it *dominates* a lecture, and that is the intended nudge:
a prof's class should sound like the prof, not like Bear.

**Register — Lecture** *(proposed — review)*. The voice of a good instructor in a
course, not a YouTuber. Define terms precisely on first use; tie the concept to the
course's learning objective ("this is the idea behind Assignment 3 / the exam question
on X"); cite the primary source or assigned reading where one exists; be rigorous and
assessable rather than punchy. Honest about limits, like all vox registers — but
course-anchored, so a student can map the reel onto what they'll be graded on. No hype,
sentence case, per Northeastern brand voice.

**Palette — `neu`** (`DESIGN.md`). Northeastern brand: **NU Red `#C8102E` = brand /
emphasis / primary series only, never state**; black `#000000` text; white `#FFFFFF`
ground; NU gold `#A4804A` ceremonial, large-area only. Because red cannot encode "bad,"
**good/bad is carried by label + position only** — no color-coding of state. Charts use
NU Red primary + grays, never a categorical rainbow. Typeface is **Lato** (required),
regular-weight headings — not EB Garamond/Montserrat. This is Northeastern brand law,
not a preference: every asset must contain red, gold is rare, no gradients or drop
shadows, white ground only.

**Signature tangent — Coursework tangent** *(proposed — review)*. When (and only when) a
clear moment appears to connect the concept to the course itself — an assigned reading,
a problem set, what the assessment actually tests — NEU may take one bounded zero-to-one
aside that names it ("this is the reading by X"; "you'll use this on the midterm"), then
hands back with a re-entry cue. Fits the audience (a student inside a course) and the
Lecture register (course-anchored, not a sermon). No clear course hook → it stays
silent. Capped at 0–1 per video, under the same tangent discipline as the others.

**Pedagogy.** The vox core holds — concrete before abstract, open on a genuine mystery,
explain the mechanism from first principles, name the trade-off, be honest about limits.
NEU adds one constraint: it is **course-aware**. It assumes syllabus context, names the
learning objective, and uses terminology precisely enough to be exam-relevant. Teach to
be assessed, not just to be watched.
